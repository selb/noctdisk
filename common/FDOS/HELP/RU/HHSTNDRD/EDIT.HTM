edit

Запуск DOS редактора. 

  edit [[d:][путь]файл] [опции]

Опции


  [[d:]путь]файл]

  Укажите файл(ы) для открытия. Можно использовать маски * и ? .

  /B

  Используйте этот режим для чёрно-белых (монохромных) дисплеев.

  /H

  Задействовать режим высокого видео/текст разрешения.

Дополнительная информация

Edit имеет встроенную очень содержательную систему попощи.

Ограничения

Edit пока не может открывать файлы, объём которых более 64кб.
